# Contributing

- Compare with `create-bunny-launcher`
- Compare with `bunny-sync`
- Update `docs/public/static/json-schema/create-bunny-launcher/schema.json`
- `corepack enable npm`
- `npm login`
- `npm install`
- `npm run prepublishOnly`
- `git add -A`
- `git commit -m "message"`
- `npm version patch`
- `git push`
- `npm publish`
